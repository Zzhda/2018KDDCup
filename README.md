# 2018KDDCup
2018 KDD CUP of Fresh Air

# 比赛简介
- 首页:[https://biendata.com/competition/kdd_2018/][1]
- 任务：通过比赛中提供的中国北京和另一个城市的数据，预测未来48小时内PM2.5, PM10和O3的浓度。例如，在5月1日，选手可以获得实时和过去的天气及空气质量数据，并预测5月2日和5月3日的污染物浓度。

# 运行环境
- 操作系统：Win10 专业版
- Python：3.6.4
- IDE: PyCharm 2017.3.4
